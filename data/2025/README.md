# ROCC 2025 Data

This directory contains the archived data for Realms of Cyber Conference 2025.

## Files

- `speakers.py` - All speakers from ROCC 2025
- `schedule.py` - Main schedule and OT schedule from ROCC 2025
- `sponsors.py` - Sponsors from ROCC 2025
- `exhibitors.py` - Exhibitors from ROCC 2025
- `variables.py` - Year-specific variables (date, etc.) for ROCC 2025

## Usage

This data is automatically loaded by `year_data.py` when generating archive pages. The 2025 archive page (`2025.html`) uses this data.

## Future Years

When preparing for ROCC 2026:
1. The root-level files (`speakers.py`, `schedule.py`, etc.) will be updated with 2026 data
2. Before updating, copy the current 2026 data to a `2026/` directory (similar to this one)
3. Update `year_data.py` to include the new year
